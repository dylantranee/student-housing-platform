# HouPlatform Backend

Web application backend for finding rental houses and roommates.

## Technology Stack

- **Runtime**: Node.js
- **Package Manager**: npm

## Prerequisites

- Node.js (LTS version recommended)
- npm

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/anlecute/houplatform-be.git
cd houplatform-be
```

### Install Dependencies

```bash
npm install
```

### Run the Application

```bash
node index.js
```
